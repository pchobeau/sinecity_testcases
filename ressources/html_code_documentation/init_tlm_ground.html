init_tlm_ground module¶

init_tlm_ground.tlm_srl_init_impgr(dt, dl, h_num, h_set, d_sr, h_s, h_r, T, f_max, rho, sigma, case, free_field, disp_inst_p)[source]¶

Setting the 2D geometries and running the TLM update for case 3: ground reflection. Main script that contains all the parameters to run the TLM update in 2D.

Parameters:
  • dt (float) – time step (s).
  • dl (float) – spatial step (m).
  • h_num (int) – spatial step index.
  • h_set (list of floats) – spatial step sequence (m).
  • d_sr (list of floats) – horizontal distances between the source and the receivers (m).
  • h_s (float) – height of the source (m).
  • h_r (float) – height of the receiver (m).
  • T (float) – simulation duration (s).
  • f_max (float) – approximated maximale frequency of the source signal (Hz).
  • rho (float) – air density (kg.m-3).
  • sigma (float) – pecific airflow resistivity (kNm-4s==CGS).
  • case (int) – integer that sorts of the saved folders in the results dir.
  • free_field (bool) – the domain is enlarged
  • disp_inst_p (bool) – display the instantaneous pressure.
  • Nx – discrete length of the domain (number of node) following

the x-axis direction. :param Ny: discrete length of the domain (number of node) following the y-axis direction. :param x: discrete length sequence of a domain side, scalar (m). :param dx: spatial step after discretization, scalar (m). :param Nt: number of iteration time, scalar. :param t: discretized time sequence, 1d array (s). :param n: discrete iteration inside the for loop, scalar. :param It: discretized time sequence, 1d array. :param Ts: time step after dicretization, scalar (s).

Parameters:
  • src_dly – source delay in number of iteration, scalar.
  • x_src – discrete x coordinate of the source, int (number of node).
  • y_src – discrete y coordinate of the source, int (number of node).
  • x_rcv – discrete x coordinate of the receiver, int (number of node).
  • y_rcv – discrete y coordinate of the receiver, int (number of node).
  • Z_tlm – impedance of TLM branch, scalar.
  • Z_air – impedance of air, scalar.
  • Z_ratio – impedance ratio between tlm branch and air, scalar.
  • K – order of the recursive convolution method for the ground imp, int.
  • a_k – residuals of the partial fraction expansion, list of K size.
  • gamma_k – poles of the partial fraction expansion, list of K size.
  • I_t – incident pulse from the top.
  • I_b – incident pulse from the bottom.
  • I_l – incident pulse from the left.
  • I_r – incident pulse from the right.
  • S_t – scattered pulse from the top.
  • S_b – scattered pulse from the bottom.
  • S_l – scattered pulse from the left.
  • S_r – scattered pulse from the right.
  • psi_k – impedance accumulator at n+1 for the recursive convolution,

numpy array (dimension of the scene). :param psi1_k: impedance accumulator at n for the recursive convolution, numpy array (dimension of the scene).

Parameters:geo – boundary of the domain (1 if BC, 0 else), similar to Nb in FDTD scripts, numpy array (dimension of the scene).
Returns:the acoustic pressure at the pre-defined receivers’ locations as a

function of time. :rtype: (1+1)D array of floats 64
