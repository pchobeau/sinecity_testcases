pe_att_spectrum module¶

pe_att_spectrum.attenuation_spectrum(rho, c, h_s, h_r, d_sr, freq, sigma, case)[source]¶

Show the attenuation spectrums along the frequency range defined in the main module for the PE simulations. The PE results are compared to the analytic solution that is calculated in the present function using the simulations parameters.

Parameters:
  • rho (float) – air density (kg.m-3).
  • c (float) – sound speed (m.s-1).
  • h_s (float) – height of the source (m).
  • h_r (float) – height of the receiver (m).
  • d_sr (list of floats) – horizontal distances between the source and the receivers (m).
  • freq (1darray of floats) – frequency of the simulation (Hz).
  • sigma (list of int) – specific airflow resistivity (kNm-4s==CGS).
  • case (int) – integer that sorts of the saved folders in the results dir.
Returns:

plots of the attenuation spectrums
