pe_absorbing_layer module¶

pe_absorbing_layer.abs_lay_bottom_top(p_ij, dy, Ny, y_start_abs)[source]¶

Absorbing layer for the ground (low part) of the parabolic equation, in order to simulate free field propagation. The absorbing layer is defined following the vertical direction - i.e. along the y axis. For more details see [chevret_phd1994, Eq.(4.38), p.59].

Parameters:
  • p_ij (2D numpy arrays of complexes) – pressure at the discrete location i,j ~ (x, y) (Pa).
  • dy (float) – spatial step for the y directions (m).
  • Ny (int) – length of the domain in number of nodes following the y dir.
  • y_start_abs (float) – y coordinate of the layer starting position (m).
Returns:

the pressure array inclunding the absorption of the layer (Pa).
Return type:

2D numpy arrays of complexes
