init_tlm_modes module¶

init_tlm_modes.tlm_srl_init_conv_box(dl, h_num, h_set, t_save, T, Lx, Ly, nx, ny, fr, rho, c, case, disp_inst_p)[source]¶
Setting the 2D geometries and running the TLM update for case 2: acoustic modes.

Main script that contains all the parameters to run the TLM update in 2D.

Parameters:
  • dl (float) – spatial step (m).
  • h_num (int) – spatial step index.
  • h_set (list of floats) – spatial step sequence (m).
  • t_save (float) – time instant at which the pressure maps are saved for comparison (s).
  • T (float) – simulation duration (s).
  • Lx (float) – length of the box followinf the x axis (m)
  • Ly (float) – length of the box followinf the y axis (m)
  • nx (int) – mode number following the x direction
  • ny (int) – mode number following the y direction
  • fr –
  • rho (float) – air density (kg.m-3).
  • c (float) – sound speed (m.s-1).
  • case (int) – integer that sorts of the saved folders in the results directory.
  • disp_inst_p (bool) – display the instantaneous pressure.

:param Nx discrete length of the domain (number of node) following the x-direction. :param Ny discrete length of the domain (number of node) following the y-direction. :param x discrete length sequence of a domain side, scalar (m). :param dx spatial step after discretization, scalar (m). :param Nt number of iteration time, scalar. :param t discretized time sequence, 1d array (s). :param n discrete iteration inside the for loop, scalar. :param It discretized time sequence, 1d array. :param Ts time step after dicretization, scalar (s).

:param src_dly source delay in number of iteration, scalar.

:param I_t incident pulse from the top. :param I_b incident pulse from the bottom. :param I_l incident pulse from the left. :param I_r incident pulse from the right.

:param geo boundary of the domain (1 if BC, 0 else), similar to Nb
in FDTD scripts, numpy array (dimension of the scene).
Returns:the acoustic pressure at the pre-defined receivers’ locations as a function of time.
Return type:(1+1)D array of floats 64
