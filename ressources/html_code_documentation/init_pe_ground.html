init_pe_ground module¶

init_pe_ground.pe_init_impgr(delta_x, d_sr, h_s, h_r, freq, rho, c, sigma, case, free_field, disp)[source]¶

Definition of the numerical domain for the application of the parabolic equation to the ground reflection case, with an impedance ground defined using Miki’s impedance model.

Parameters:
  • delta_x (float) – spatial step for the x directions (m).
  • d_sr (list of floats) – horizontal distances between the source and the receivers (m).
  • h_s (float) – height of the source (m).
  • h_r (float) – height of the receiver (m).
  • freq (float) – frequency of interest (Hz).
  • rho (float) – air density (kg.m-3).
  • c (float) – sound speed (m.s-1).
  • sigma (float) – pecific airflow resistivity (kNm-4s==CGS).
  • case (int) – integer that sorts of the saved folders in the results dir.
  • free_field (bool) – the domain is enlarged
  • disp (bool) – display the pressure inside the numerical domain.
Returns:

plot and/or save the pressure arrays for post processing.
