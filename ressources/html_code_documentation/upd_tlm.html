TLM update with the impedance boundary condition shown in [guillaume_jsv2011]. Diffusion matrix, connexion laws and boundary condition for 2D TLM network.

  • rho (float) – air density (kg.m-3).
  • c (float) – sound speed (m.s-1).
  • Ts (float) – time step (s).
  • Z_air (complex) – impedance of air
  • Z_tlm (complex) – impedance of TLM branch
  • Z_ratio (complex) – impedance ratio between tlm branch and air
  • K (int) – order of the recursive convolution method for the ground imped.
  • a_k (list of floats) – residuals of the partial fraction expansion
  • gamma_k (list of floats) – poles of the partial fraction expansion
  • I_t (2D numpy array of floats.) – incident pulse from the top
  • I_b (2D numpy array of floats.) – incident pulse from the bottom
  • I_l (2D numpy array of floats.) – incident pulse from the left
  • I_r (2D numpy array of floats.) – incident pulse from the right
  • S_t (2D numpy array of floats.) – scattered pulse from the top
  • S_b (2D numpy array of floats.) – scattered pulse from the bottom
  • S_l (2D numpy array of floats.) – scattered pulse from the left
  • S_r (2D numpy array of floats.) – scattered pulse from the right
  • psi_k (2D numpy array of floats.) – impedance accumulator at time n+1 for the recursive convol.
  • psi1_k (2D numpy array of floats.) – impedance accumulator at time n for the recursive convol.
Returns:

the updated pressure at the time n+1, variable p.

2D numpy array of floats.
