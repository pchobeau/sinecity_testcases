pe_source module¶

pe_source.gaussian_source(k, h, h_s, Nx, Ny)[source]¶

Spatial Gaussian source that is imposed on the first row of the domain, see - [chevret_phd1994, Eqs.(4.37), p.59] or

[blairon_phd2002, Eqs.(2.21)-(2.25)].
Parameters:
  • k (float) – wave number: k=2*np.pi*f/c0 (rad.m-1).
  • h (float) – spatial step for both Cartesian directions (m).
  • h_s (float) – height of the source (m).
  • Rp (float) – reflection coefficient of the ground.
  • Nx (int) – length of the domain in number of nodes following the x dir.
  • Ny (int) – length of the domain in number of nodes following the y dir.
Returns:

the source pressure to be imposed at the entrance of the domain.
Return type:

1D array of floats
pe_source.gaussian_source_imp(k, beta, h, h_s, Nx, Ny)[source]¶

Spatial Gaussian source that is imposed on the first row of the domain. It includes an image source pondered by the ground reflection coefficient, see [blairon_phd2002, Eqs.(2.36)].

Parameters:
  • k (float) – wave number: k=2*np.pi*f/c0 (rad.m-1).
  • beta – NORMALIZED admitance used for the boundary condition (kg.s.m-2).
  • h (float) – spatial step for both Cartesian directions (m).
  • h_s (float) – height of the source (m).
  • Rp (float) – reflection coefficient of the ground.
  • Nx (int) – length of the domain in number of nodes following the x dir.
  • Ny (int) – length of the domain in number of nodes following the y dir.
Returns:

the source pressure to be imposed at the entrance of the domain.
Return type:

1D array of floats
