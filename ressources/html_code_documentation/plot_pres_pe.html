plot_pres_pe module¶

plot_pres_pe.plot_pressure_pe(p, h_s, freq, dx, dy)[source]¶

Display the absolute pressure in the 2D domain of the parabolic equation method. The pressure can be pre-processed to show relative or absolute levels, or just in Pascal directly from the init_pe_*.

Parameters:
  • p (2D numpy arrays of complexes) – pressure from the parabolic equation solver (Pa, dB…).
  • h_s (float) – height of the source (m).
  • freq (float) – frequency of interest (Hz).
  • dx (float) – spatial step for the x directions (m).
  • dy (float) – spatial step for the y directions (m).
Returns:
Return type:

SineCity
