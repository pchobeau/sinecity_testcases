upd_pe module¶

upd_pe.pe_pade_one_one(k, beta, dx, dy, Nx, Ny)[source]¶

Initialization of the matrices for the parabolic equation, in the case of acoustic wave propagation (Helmholtz) in an homogeneous atmosphere - i.e. no temperature, no wind gradients, using the split-step Pade method. This function corresponds to the Pade(1, 1) expansion. Matrices are defined using 2D numpy arrays that contain the complex coef.

Parameters:
  • k (float) – wave number: k=2*np.pi*f/c0 (rad.m-1).
  • beta – NORMALIZED admitance used for the bound. cond. (kg.s.m-2).
  • dx (float) – spatial step for the x directions (m).
  • dy (float) – spatial step for the y directions (m).
  • Nx (int) – length of the domain in number of nodes following the x dir.
  • Ny (int) – length of the domain in number of nodes following the y dir.
Returns:

the updated pressure vetcor at the location i + 1
Return type:

2D numpy arrays of complexes

upd_pe.pe_scheme_pade_1_1(k, beta, dx, dy, p_ij, Nx, Ny)[source]¶

It contains both the array filling and the solver in the same function. Parabolic equation scheme that solves the acoustic wave propagation eq. (Helmholtz) in an homogeneous atmosphere - i.e. no temperature, no wind gradients, using the split-step Pade method order (1, 1). This module calculates the main matrices according to [blairon_phd2002, Eqs.(2.18)-(2.19), p.45]. Matrices are defined using 2D numpy arrays that contain the complex coef.

Parameters:
  • k (float) – wave number: k=2*np.pi*f/c0 (rad.m-1).
  • beta – NORMALIZED admitance used for the bound. cond. (kg.s.m-2).
  • dx (float) – spatial step for the x directions (m).
  • dy (float) – spatial step for the y directions (m).
  • p_ij (2D numpy arrays of complexes) – pressure at the discrete location i,j ~ (x, y) (Pa).
  • Nx (int) – length of the domain in number of nodes following the x dir.
  • Ny (int) – length of the domain in number of nodes following the y dir.
Returns:

the updated pressure array
Return type:

2D numpy arrays of complexes

upd_pe.pe_solver(Aij, Bij, pi)[source]¶

Parabolic equation solver that calculates the matrix product and the matrix inversion for each spatial iteration follwing the x direction (i +1 <- i).

Parameters:
  • Aij (2darray of complexes) – matrix from the LHS of [chevret_phd1994, Eqs.(4.69), p.70].
  • Bij (2darray of complexes) – matrix from the RHS of [chevret_phd1994, Eqs.(4.69), p.70].
  • pi (1darrays of complexes (vector)) – pressure at row i (Pa).
Returns:

pressure at row i + 1 (Pa).
