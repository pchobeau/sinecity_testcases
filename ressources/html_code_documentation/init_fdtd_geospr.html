init_fdtd_geospr module¶

init_fdtd_geospr.fdtd_srl_init_conv(dl, h_num, dt, d_sr, T, f_max, rho, c, case, disp_inst_p)[source]¶

Setting the 2D geometries and running the FDTD update for case 1: geometrical spreading. Main script that contains all the parameters to run the FDTD update in 2D.

Parameters:
  • dl (float) – spatial step (m).
  • h_num (int) – spatial step index.
  • dt (float) – time step (s).
  • d_sr (list of floats) – horizontal distances between the source and the receivers, list of floats (m).
  • T (float) – simulation duration (s).
  • f_max (float) – approximated maximale frequency of the source signal (Hz).
  • rho (float) – air density (kg.m-3).
  • c (float) – sound speed (m.s-1).
  • case (int) – integer that sorts of the saved folders in the results directory.
  • disp_inst_p (bool) – display the instantaneous pressure.
  • Lx – continuous length of the domain (in meter) following the x-direction.
  • Ly – continuous length of the domain (in meter) following the y-direction.
  • Nx – discrete length of the domain (number of node) following the x-direction.
  • Ny – discrete length of the domain (number of node) following the y-direction.
  • x – discrete length sequence of a domain side, scalar (m).
  • dx – spatial step after discretization, scalar (m).
  • Nt – number of iteration time, scalar.
  • t – discretized time sequence, 1d array (s).
  • It – discretized time sequence, 1d array.
  • Ts – time step after dicretization, scalar (s).
  • Cn – Courant number, scalar.
  • p – updated pressure (n+1), numpy array (dimension of the scene).
  • p1 – current pressure (n), numpy array (dimension of the scene).
  • p2 – past pressure (n-1), numpy array (dimension of the scene).
  • fsrc – soft source (n+1), numpy array (dimension of the scene).
  • fsrc1 – soft source (n), numpy array (dimension of the scene).
  • p_axial – pressure saved on the axial axis, 2d array (number of recievers * n).
  • p_diag – pressure saved on the diagonal axis, 2d array (number of recievers * n).
  • A – inertance of the boundary, scalar.
  • B – stiffness of the boundary, scalar.
  • C – resistivity of the boundary, scalar.
  • Nb – boundary of the domain (1 if BC, 0 else) for the compact pressure update, numpy array (dimension of the scene).
  • x_src – discrete x coordinate of the source, scalar (number of node).
  • y_src – discrete y coordinate of the source, scalar (number of node).
  • x_rcv – discrete x coordinate of the receiver, scalar (number of node).
  • y_rcv – discrete y coordinate of the receiver, scalar (number of node).
  • n – discrete iteration inside the for loop, scalar.
Returns:

the acoustic pressure at the pre-defined receivers’ locations as a function of time.
Return type:

(1+1)D array of floats 64
