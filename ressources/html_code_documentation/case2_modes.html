case2_modes module¶

case2_modes.main(Lx, Ly, nx, ny, fr)[source]¶

Each method (FDTD or TLM) is called using the init_*.py functions. The resutls are post-processed using error_calc function.

Parameters:
  • Lx (float) – length of the box followinf the x axis (m)
  • Ly (float) – length of the box followinf the y axis (m)
  • nx (int) – mode number following the x direction
  • ny (int) – mode number following the y direction
  • fr (float) – frequency of the sine source signal (Hz)
